# Ishwar Trading Mobile App

A Flutter-based business management application developed for **Shri Ishwar Trading Company**.  
The application is designed to manage and streamline the complete trading workflow including:

- Sale Confirmation / SaudaPatrak
- Dispatch Management
- Invoice Management
- Payment Tracking
- Commission Management
- Commission Bills
- Commission Payments
- Reports & Document Generation

---

# Project Overview

This project is being developed by referencing the existing **Divisha SaudaPatrak** system while customizing the workflow and document structure according to the requirements of **Shri Ishwar Trading Company**.

The application focuses on:
- Simplified Sale Confirmation workflow
- Clean and printable document generation
- GST-ready transaction management
- Dispatch and payment tracking
- Modular business operations
